# Guía de Contribución

¡Gracias por tu interés en contribuir a este proyecto! 

Este es un proyecto académico, pero seguimos buenas prácticas de desarrollo. Si deseas colaborar, por favor sigue estos pasos:

## Flujo de Trabajo

1.  **Fork** del repositorio.
2.  Crea una nueva rama para tu funcionalidad (`git checkout -b feature/AmazingFeature`).
3.  Realiza tus cambios (asegúrate de que el código compila y sigue el estilo del proyecto).
4.  Haz **Commit** de tus cambios (`git commit -m 'Add some AmazingFeature'`).
5.  Haz **Push** a la rama (`git push origin feature/AmazingFeature`).
6.  Abre un **Pull Request** hacia la rama `master`.

## Normas de Código

* Usa **TypeScript** estricto siempre que sea posible.
* Mantén los componentes de Next.js pequeños y reutilizables.
* Asegúrate de no subir credenciales o archivos `.env`.

## Reportar Errores

Si encuentras un bug, por favor abre un **Issue** describiendo:
* El error.
* Pasos para reproducirlo.
* Comportamiento esperado.
